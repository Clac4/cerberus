* ssh config

tangle files over ssh with emacs
edit you ~/.ssh/config and add the following code

#+NAME: ssh
#+begin_src conf
# emacs tramp ssh key
Host *
IdentityFile ~/.ssh/id_rsa.pub
ServerAliveInterval 5
#+end_src
